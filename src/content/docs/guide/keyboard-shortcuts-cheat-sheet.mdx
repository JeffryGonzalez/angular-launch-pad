- **Delete Word Backward**: <KeyboardShortcut keys="Ctrl+W" />
- **Delete to End of Line**: <KeyboardShortcut keys="Ctrl+K" />

### Common Commands

```bash
pwd              # Print working directory
ls               # List files
ls -la           # List all files (including hidden) with details
cd directory     # Change directory
cd ..            # Go up one level
cd ~             # Go to home directory
mkdir name       # Create directory
rm file          # Remove file
rm -rf directory # Remove directory and contents (use carefully!)
cp source dest   # Copy file
mv source dest   # Move/rename file
```

## Tips for Learning Shortcuts

1. **Start Small**: Pick 3-5 shortcuts you'll use most often and practice them deliberately
2. **Command Palette is Your Friend**: When you forget a shortcut, use the Command Palette (`Cmd/Ctrl+Shift+P`) - it shows shortcuts next to commands
3. **Print This Out**: Keep a printed copy near your workstation during the first week
4. **Use Them Daily**: The only way shortcuts become muscle memory is through repetition
5. **Customize if Needed**: If a shortcut conflicts with your OS or doesn't feel natural, remap it in VS Code settings
